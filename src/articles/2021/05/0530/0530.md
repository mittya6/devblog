# インストール
とりあえず、はいはいインストールでOKだが、
「pythonを環境変数に追加する」というのはチェックしておこう。
してなかったら、もう一度インストーラを実行してModifyで設定しよう。

```cmd
python -V

```

# ファイル作成する。

