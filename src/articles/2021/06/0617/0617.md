# Vue 3をインストール
```javascript
npm init -y
npm install @vue/cli
```

バージョン確認
```javascript
npx vue --version
@vue/cli 4.5.13
```

# プロジェクト作成
```javascript
npx vue create my-project
```

# ブラウザから動かす
```javascript
cd my-project
npm run serve
```


# ビルドする。
```javascript
npm run build
```
